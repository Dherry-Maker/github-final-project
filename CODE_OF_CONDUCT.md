# Code of Conduct

## Project: github-final-project

Maintained by **Dherry-Maker**

---

## Our Pledge

As contributors and maintainers of this student project, we pledge to make participation in
**githubfinalproject** a respectful, welcoming, and harassment-free experience for everyone.

This project is intended for learning, collaboration, and growth.

---

## Expected Behavior

Participants are expected to:

- Be respectful and kind to others
- Use constructive and polite language
- Give helpful feedback and accept criticism professionally
- Respect different skill levels and learning paths
- Follow basic academic and professional integrity

---

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, insults, or discriminatory language
- Trolling or intentionally disruptive behavior
- Personal attacks or threats
- Sharing private information without permission
- Any behavior that negatively affects a positive learning environment

---

## Responsibilities

The project maintainer (**Dherry-Maker**) is responsible for:

- Clarifying acceptable standards of behavior
- Addressing inappropriate behavior fairly and respectfully
- Removing or rejecting contributions that violate this Code of Conduct

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repositories
- Issues and pull requests
- Discussions and comments
- Any communication related to this project

---

## Enforcement

If unacceptable behavior occurs, the maintainer may take actions such as:

- Giving a warning
- Requesting changes or apologies
- Temporarily or permanently restricting participation

All actions will be taken with fairness and respect.
